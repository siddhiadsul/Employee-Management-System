# Employee Management System - Java

## Description
The Employee Management System is a console-based Java application that allows users to manage employee records using Object-Oriented Programming principles. The system supports basic CRUD operations such as adding, viewing, updating, and deleting employee details using in-memory storage.

## Features
- Add new employees with ID, name, and salary
- View all existing employees
- Update employee details using employee ID
- Delete employee records using employee ID
- Menu-driven and user-friendly console interface
- Uses ArrayList for dynamic data storage

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner class
- Console-based application

## How to Run the Project
1. Open the project in IntelliJ IDEA or Eclipse
2. Navigate to `EmployeeManagementSystem.java`
3. Run the file
4. Choose options from the menu to perform operations:
    - Add Employee
    - View Employees
    - Update Employee
    - Delete Employee
    - Exit
